# Verwendete Prompts in Google Stitch

## Prompt 1 – Erster Entwurf

```text
Create a simple and modern web prototype for a university student task manager called “Smart Task Manager”.

The main screen should contain:

- A header with the title “Smart Task Manager”
- A clear button called “Create Task”
- Three small summary cards:
  - Open Tasks
  - Completed Tasks
  - Overdue Tasks
- A search field
- A filter for priority
- Sorting by priority or due date
- A clear list of tasks

Each task should show:

- Title
- Short description
- Priority: HIGH, MEDIUM or LOW
- Status: NEW or COMPLETED
- Due date
- A “Complete” button
- A “Delete” button

Also create a simple form for creating a new task with:

- Title
- Description
- Priority
- Due date
- Save button
- Cancel button

Use a clean blue, white and light gray design.
The design should be responsive for desktop and mobile.
Keep it simple and beginner-friendly.

Do not add login, registration, chat, calendar integration, task editing or other extra features.
```

## Prompt 2 – Vereinfachter Entwurf

```text
Create a simple web interface for a Smart Task Manager for students.

The design should be clean, minimal, and beginner-friendly.
Do not create a complex dashboard.

Please include only these elements:
- a top bar with the title "Smart Task Manager"
- one button "Add Task"
- a simple task form with:
  - title
  - description
  - priority
  - due date
- a task list showing existing tasks
- simple actions for each task:
  - mark as completed
  - delete
- a small filter area:
  - all
  - open
  - completed

Optional:
- a very small summary row with:
  - open tasks
  - completed tasks
  - overdue tasks

Please avoid:
- calendar pages
- course pages
- settings pages
- large analytics
- side navigation with many items
- complicated widgets

The result should look like a simple desktop web app that can be implemented easily with Next.js.
```

## Prompt 3 – Create-Task-Ansicht

```text
Add a second web screen called “Create Task” to the existing Smart Task Manager design.

Keep the same simple visual style as the current dashboard.

The screen should contain only:

- Page title: “Create Task”
- Input field for task title
- Text area for description
- Priority selection with HIGH, MEDIUM and LOW
- Due date field
- Primary button: “Save Task”
- Secondary button: “Cancel”

Do not add login, categories, attachments, reminders, task editing or other extra features.

Do not replace or redesign the existing dashboard.
The “Add Task” button on the dashboard should lead to this new screen.
```
